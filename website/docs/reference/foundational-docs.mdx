# Foundational Docs

The narrative docs teach Transit in a gradual order. This section publishes the canonical repo contracts beside that narrative so you can move from overview to exact wording without leaving the docs site.

:::note Source of truth
The contract pages under `Reference` are generated from Markdown files in the repository root. Edit the root files first, then run `just docs-sync` or `npm --prefix website run sync:foundations`.
:::
